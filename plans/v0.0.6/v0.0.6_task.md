# v0.0.6 Task Breakdown

## Overview
Add custom command alias feature to allow users to define and run custom command sequences via `flc run <alias>`.

---

## Phase 1 — Config Schema Extension
**Priority**: HIGH

### Tasks
- [ ] Extend `BuildConfig` class to support `alias` section
- [ ] Add `CommandAlias` model class
- [ ] Update YAML parsing to read `alias` map
- [ ] Add validation for alias names and commands
- [ ] Update `gen_command.dart` to include alias section in generated config

---

**Priority**: HIGH

### Tasks
