# v0.0.2 Subtask: Remove Arrow Mode

## Problem
Arrow mode (`stdin.lineMode = false`) fails on Windows PowerShell/CMD with:
```
StdinException: Error setting terminal line mode, errno = 87
```

Standard Dart doesn't support cross-platform raw terminal input.

## Solution
Remove arrow mode entirely. Keep only **numeric mode** which works everywhere.
